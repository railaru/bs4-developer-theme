PROJECT:

Dynamic Visualization of GLPT's '4X - DAT'

Find out more ▶

ABOUT:

Initially built for a private client as my CS personal project for the International Baccalaureate Programme, this tool is now available for every 4X-DAT user. It bridges the world of sophisticated trading algorithms with retail investing accessible to everyone.
4xdat's default UI only provides user with basic trading record keeping capabilities.
As requested by client, the tool takes data for open and close prices of trades taken by 4xdat software and plots them on a chart.
User's trades are now visualized on the MetaTrader4 platform.
Custom combinations of technical indicators can be added to the chart.
