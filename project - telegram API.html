PROJECT:

Telegram API - Trading Alerts

Find out more ▶

ABOUT:

This project takes the HVF Indicator one step further by completely automatizing pattern recognition process. Once pattern is recognized on a financial instrument chart, a message is sent to the user using the Telegram API.
The tool recognizes the pattern automatically and draws the pattern on the chart.
